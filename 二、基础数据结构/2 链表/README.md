## 本节导读

這組筆記由原本的 `2 链表.md` 拆分而來。內容先建立鏈表的節點引用模型，再依序整理單向鏈表、帶哨兵鏈表、雙向鏈表、環形鏈表，最後用多道 LeetCode 題解練習反轉、刪除、合併、找中點、判斷回文與判斷環。

第一次閱讀時建議先讀概念與基礎實作，再進入題解；複習時可以直接依題號或操作類型跳到對應小節。

## 快速回查

- 忘記鏈表分類、哨兵節點與複雜度：看 [概述](./01%20概述.md)
- 要實作單向鏈表的新增、遍歷、查找、插入、刪除：看 [单向链表](./02%20单向链表.md)
- 想理解虛擬頭節點如何簡化邊界判斷：看 [单向链表（带哨兵）](./03%20单向链表（带哨兵）.md)
- 要維護 `prev` / `next` 雙向連結：看 [双向链表（带哨兵）](./04%20双向链表（带哨兵）.md)
- 想確認首尾閉環的實作：看 [环形链表（带哨兵）](./05%20环形链表（带哨兵）.md)
- 要複習反轉鏈表：看 [LeetCode 206 反转链表](./06%20LeetCode%20206%20反转链表.md)
- 要複習刪除節點類題目：看 [LeetCode 203 移除链表元素](./07%20LeetCode%20203%20移除链表元素.md)、[LeetCode 19 刪除鏈表的倒數第 N 個節點](./08%20LeetCode%2019%20刪除鏈表的倒數第%20N%20個節點.md)、[LeetCode 82 删除排序链表中的所有重复元素 II](./10%20LeetCode%2082%20删除排序链表中的所有重复元素%20II.md)、[LeetCode 83 删除排序链表中的重复元素](./09%20LeetCode%2083%20删除排序链表中的重复元素.md)
