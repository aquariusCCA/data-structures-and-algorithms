## 本章导读

本章從數組開始，逐步學習鏈表、隊列、棧、堆、樹等基礎資料結構。建議先掌握連續存儲與節點引用的差異，再進入隊列、棧等抽象資料型別，最後再看樹與平衡樹。

第一次閱讀時建議按檔名前的序號順讀；複習時可以直接依資料結構名稱回查。
