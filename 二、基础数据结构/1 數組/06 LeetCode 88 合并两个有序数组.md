- [88. 合并两个有序数组](https://leetcode.cn/problems/merge-sorted-array/description/)

> 這題是從 **Leetcode 88. Merge Sorted Array** 延伸出的變形。
> * 和原題不同的是：
> * **不是兩個獨立陣列**，而是**同一個陣列中有兩段已排序區間**，現在要把它們合併起來。

## 題目概念

將數組兩個區間內的有序元素合併

例：

```java
[1, 5, 6, 2, 4, 10, 11]
```

它其實可以看成兩個已經排好序的區間：

```java
[1, 5, 6] 和 [2, 4, 10, 11]
```

目標是把這兩段有序區間合併成一個完整有序序列：

```java
[1, 2, 4, 5, 6, 10, 11]
```

## 方法一：遞迴版

### 核心思路

這題本質上就是：

> **合併兩個有序序列**

做法是同時用兩個指標：

* `i`：指向第一段有序區間的目前元素
* `j`：指向第二段有序區間的目前元素
* `k`：指向結果陣列 `a2` 要放入元素的位置

每次比較 `a1[i]` 和 `a1[j]`：

* 較小者放入 `a2[k]`
* 對應的指標往後移動
* `k` 也往後移動

直到某一段用完，再把另一段剩餘元素整段複製過去。

### 遞迴過程示意

偽碼如下：

```java
merge([1,5,6] [2,4,10,11], a2=[]) {
    merge([5,6] [2,4,10,11], a2=[1]) {
        merge([5,6] [4,10,11], a2=[1,2]) {
            merge([5,6] [10,11], a2=[1,2,4]) {
                merge([6] [10,11], a2=[1,2,4,5]) {
                    merge([] [10,11], a2=[1,2,4,5,6]) {
                         merge([], [], a2=[1,2,4,5,6,10,11])
                    }
                }
            }
        }
    }
}
```

> **遞迴的本質：每一層遞迴只做一件事**
> 1. 比較兩個區間目前的起點元素
> 2. 取較小的那個放進 `a2`
> 3. 縮小對應區間
> 4. 繼續合併剩下的部分

### 終止條件

遞迴停止有兩種情況：

##### 1. 第一段區間已經用完

```java
if (i > iEnd)
```

代表第一段沒元素了，直接把第二段剩餘元素整段複製到 `a2`。


##### 2. 第二段區間已經用完

```java
if (j > jEnd)
```

代表第二段沒元素了，直接把第一段剩餘元素整段複製到 `a2`。

### 程式碼

```java
public class LeetCode88 {

  public static void main(String[] args) {
    int[] a1 = {1, 5, 6, 2, 4, 10, 11};
    int[] a2 = new int[a1.length];

    // 合併 a1[0..2] 和 a1[3..6]
    merge(a1, 0, 2, 3, 6, a2, 0);

    System.out.println("合併後的暫存陣列 a2: " + Arrays.toString(a2));

    // 將結果複製回原陣列
    System.arraycopy(a2, 0, a1, 0, a2.length);
    System.out.println("複製回原陣列 a1: " + Arrays.toString(a1));
  }

  /**
   * 將 a1 中兩個有序區間合併到 a2
   *
   * @param a1   原始陣列
   * @param i    第一個有序區間起點
   * @param iEnd 第一個有序區間終點
   * @param j    第二個有序區間起點
   * @param jEnd 第二個有序區間終點
   * @param a2   結果陣列
   * @param k    a2 的目前填入位置
   */
  public static void merge(int[] a1, int i, int iEnd,
                           int j, int jEnd, int[] a2, int k) {

    // 情況1：第一段已合併完，直接複製第二段剩餘元素
    if (i > iEnd) {
      System.arraycopy(a1, j, a2, k, jEnd - j + 1);
      return;
    }

    // 情況2：第二段已合併完，直接複製第一段剩餘元素
    if (j > jEnd) {
      System.arraycopy(a1, i, a2, k, iEnd - i + 1);
      return;
    }

    // 比較兩段目前元素，較小者放入 a2
    if (a1[i] < a1[j]) {
      a2[k] = a1[i];
      merge(a1, i + 1, iEnd, j, jEnd, a2, k + 1);
    } else {
      a2[k] = a1[j];
      merge(a1, i, iEnd, j + 1, jEnd, a2, k + 1);
    }
  }
}
```

### 執行流程範例

原始資料：

```java
a1 = [1, 5, 6, 2, 4, 10, 11]
```

區間：

* 第一段：`a1[0..2] = [1, 5, 6]`
* 第二段：`a1[3..6] = [2, 4, 10, 11]`


##### 第 1 次比較

```java
a1[i] = 1, a1[j] = 2
```

`1 < 2`，所以放 `1` 到 `a2[0]`

```java
a2 = [1, _, _, _, _, _, _]
```


##### 第 2 次比較

```java
a1[i] = 5, a1[j] = 2
```

`2 < 5`，所以放 `2` 到 `a2[1]`

```java
a2 = [1, 2, _, _, _, _, _]
```

##### 第 3 次比較

```java
a1[i] = 5, a1[j] = 4
```

`4 < 5`，所以放 `4` 到 `a2[2]`

```java
a2 = [1, 2, 4, _, _, _, _]
```

##### 第 4 次比較

```java
a1[i] = 5, a1[j] = 10
```

`5 < 10`，所以放 `5`

```java
a2 = [1, 2, 4, 5, _, _, _]
```

##### 第 5 次比較

```java
a1[i] = 6, a1[j] = 10
```

`6 < 10`，所以放 `6`

```java
a2 = [1, 2, 4, 5, 6, _, _]
```

##### 第一段已用完

剩下第二段：

```java
[10, 11]
```

直接複製過去：

```java
a2 = [1, 2, 4, 5, 6, 10, 11]
```

## 方法二：迴圈版

這一題一樣是把同一個數組中的兩段有序區間合併起來。

例如：

```java
[1, 5, 6, 2, 4, 10, 11]
```

可以拆成兩個有序區間：

```java
[1, 5, 6] 和 [2, 4, 10, 11]
```

合併後得到：

```java
[1, 2, 4, 5, 6, 10, 11]
```

### 核心思路

方法二不用遞迴，而是改用 **while 迴圈** 來完成合併。

準備三個指標：

* `i`：指向第一個有序區間的目前元素
* `j`：指向第二個有序區間的目前元素
* `k`：指向結果陣列 `a2` 目前要填入的位置

每次比較 `a1[i]` 和 `a1[j]`：

* 誰比較小，就放進 `a2[k]`
* 對應的指標往後移動
* `k` 也往後移動

當其中一段區間走完後，再把另一段剩餘元素直接複製到 `a2`。

### 指標移動過程

初始狀態：

```text
i
1   5   6

j
2   4   10  11

k
```

---

第一次比較：`1` 和 `2`

`1` 較小，放入 `a2`

```text
    i
1   5   6

j
2   4   10  11

    k
1
```

---

第二次比較：`5` 和 `2`

`2` 較小，放入 `a2`

```text
    i
1   5   6

    j
2   4   10  11

        k
1   2
```

---

第三次比較：`5` 和 `4`

`4` 較小，放入 `a2`

```text
    i
1   5   6

        j
2   4   10  11

            k
1   2   4
```

---

第四次比較：`5` 和 `10`

`5` 較小，放入 `a2`

```text
        i
1   5   6

        j
2   4   10  11

                k
1   2   4   5
```

---

第五次比較：`6` 和 `10`

`6` 較小，放入 `a2`

接著第一段區間已經走完，直接把第二段剩下的 `10, 11` 複製過去：

```text
            i
1   5   6

        j
2   4   10  11

                    k
1   2   4   5   6   10  11
```

### 程式碼

```java
public class LeetCode88 {
    public static void main(String[] args) {
        int[] a1 = {1, 5, 6, 2, 4, 10, 11};
        int[] a2 = new int[a1.length];

        // 合併 a1[0..2] 和 a1[3..6]
        merge(a1, 0, 2, 3, 6, a2);

        System.out.println("合併後的暫存陣列 a2: " + Arrays.toString(a2));

        // 將結果複製回原陣列
        System.arraycopy(a2, 0, a1, 0, a2.length);
        System.out.println("複製回原陣列 a1: " + Arrays.toString(a1));
    }

    /**
     * 將 a1 中兩個有序區間合併到 a2
     *
     * @param a1   原始陣列
     * @param i    第一個有序區間起點
     * @param iEnd 第一個有序區間終點
     * @param j    第二個有序區間起點
     * @param jEnd 第二個有序區間終點
     * @param a2   暫存結果陣列
     */
    public static void merge(int[] a1, int i, int iEnd, int j, int jEnd, int[] a2) {
        int k = i;

        // 只要兩段都還沒走完，就持續比較
        while (i <= iEnd && j <= jEnd) {
            if (a1[i] < a1[j]) {
                a2[k] = a1[i];
                i++;
            } else {
                a2[k] = a1[j];
                j++;
            }
            k++;
        }

        // 如果第一段走完，複製第二段剩餘元素
        if (i > iEnd) {
            System.arraycopy(a1, j, a2, k, jEnd - j + 1);
        }

        // 如果第二段走完，複製第一段剩餘元素
        if (j > jEnd) {
            System.arraycopy(a1, i, a2, k, iEnd - i + 1);
        }
    }
}
```
