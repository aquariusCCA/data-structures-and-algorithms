# 一、初识算法

本章主要建立演算法入門所需的基礎概念：如何衡量算法好壞、如何理解二分查找，以及如何讀懂遞歸。

這一章適合在進入基礎資料結構之前先順讀，後續遇到複雜度分析、查找邏輯或遞歸題型時，也可以回來查。

## 本章主要內容

- 用時間複雜度與空間複雜度衡量算法好壞
- 二分查找的基本寫法、邊界條件與常見變形
- 遞歸的執行流程、設計思路與基礎練習題
